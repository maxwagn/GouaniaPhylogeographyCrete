# CretePopGenomics

Code and analysis pipelines accompanying the manuscript:

"Concordant connectivity and parallel genomic divergence, but heterogeneous demographic history, in sympatric Mediterranean clingfishes"

## Repository structure

- `scripts/` : custom analysis scripts
- `notebooks/` : Jupyter notebooks used for analyses and visualization
- `snakemake_pipelines/` : Snakemake workflows

## Data availability

Raw sequencing data are available from ENA (accession numbers to be added upon publication).

Intermediate and processed data are not included in this repository.

## Reproducibility

Analyses were performed using Snakemake and Python. Software versions are documented in the corresponding workflow and environment files.
